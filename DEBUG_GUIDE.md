# 类和对象调试方法指南

## 调试技巧

### 1. 对象状态检查
使用 `getter` 方法在关键位置输出对象状态：
```cpp
cout << "Center: (" << shape->getCenter().getX() << ", " << shape->getCenter().getY() << ")" << endl;
```

### 2. 对象计数器验证
利用 `Shape::getObjectCount()` 检查内存泄漏，确保对象正确创建和销毁。

### 3. 变换操作验证
在变换前后打印对象状态，验证变换是否正确执行。

### 4. 可视化调试
使用 `draw()` 方法在图形窗口中绘制对象，直观检查变换效果。

### 5. 常见问题
- **内存泄漏**：确保所有 `new` 都有对应的 `delete`
- **空指针**：使用前检查指针是否为 `nullptr`
- **坐标溢出**：检查变换后坐标是否在窗口范围内
- **多态问题**：确保基类析构函数为虚函数
